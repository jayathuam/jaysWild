# рҹҰҒ jaysWild вҖ” Wildlife Photography Portfolio

> *Where every frame tells a story from the wild.*

A stunning, performance-obsessed wildlife photography portfolio built with **Next.js 16**, featuring a time-adaptive hero, immersive masonry galleries, and behind-the-lens storytelling. Crafted using the **BMAD methodology** from idea вҶ’ architecture вҶ’ code.

| рҹҰӢ Firefly Social Icons | вҡЎ Blazing Performance | рҹҢҗ SEO-First |

## рҹҡҖ Getting Started

```bash
# Clone the repo
git clone https://github.com/jayathu/jaysWild.git
cd jaysWild

# Install dependencies
yarn install

# Run in development (Turbopack вҡЎ)
yarn dev

# Build for production
yarn build

# Run E2E tests
yarn test

# Sync images from Google Drive
yarn sync-images
```
